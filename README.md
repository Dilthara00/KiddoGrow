## 🚀 Installation

### Prerequisites
- Java 17+
- Node.js 18+
- MongoDB 6.0+
- Maven 3.8+

```bash
# Clone repository
git clone https://github.com/yourusername/KiddoGrow-Online-Learning.git
cd KiddoGrow-Online-Learning

# Backend setup
cd backend
mvn clean install

# Frontend setup
cd ../frontend
npm install
